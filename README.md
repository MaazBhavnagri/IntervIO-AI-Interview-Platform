# IntervIO

AI-powered interview simulator that generates questions based on category or resume and evaluates answers using AI.

## Features

- AI-generated interview questions
- Resume-based interview generation
- Interview timer
- Audio and text answers
- AI evaluation with feedback
- Dashboard with history

## Tech Stack

- ASP.NET Core MVC
- C#
- OpenRouter API
- PdfPig for resume parsing
- Bootstrap UI

## How to Run

```bash
dotnet run
